# neos-gmaps
A Neos CMS package to add Google Maps as node

## Installation
Add the package in your site package composer.json

```
"require": {
  "obisconcept/neos-gmaps": "~1.0"
}
```

## Usage

Add your API key to the `Settings.yaml` of your site package

```
ObisConcept:
  NeosGmaps:
    apiKey: '...'
```
